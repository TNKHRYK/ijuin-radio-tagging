# ijuin-radio-tagging
Set ID3 tag for 『月曜JUNK深夜の馬鹿力』 MP3

## Usage

    $ python3 setTag.py [mp3 file path]

## An error occurred.
    #Permission
    $ chmod +x setTag.py

    #ModuleNotFoundError: No module named 'mutagen'
    $ pip3 install mutagen

## Result
Key | Value
---|---
album | 深夜の馬鹿力 yyyy年
artist | 伊集院光
author | 伊集院光
genre | Radio
date | yyyymmdd
